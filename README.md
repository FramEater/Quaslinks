What is Quaslinks ?
======
Quaslinks is simple CLI script that check url you enter to exploit possible vulnrabilityes.

How to use it ?
====
To use Quaslinks you must have installed latest ruby version.

To run script enter following code in terminal:

	$ ./Quaslinks
